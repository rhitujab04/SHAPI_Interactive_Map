# SHAPI_Interactive_Map

Interactive Spatial Heat Adaptation Priority Index (SHAPI) Map for Delhi NCR

This repository contains the interactive visualization used in the study _"Intensifying Urban Heat Island Hotspots and Adaptation Priorities in Delhi"_.

**Files:**

- SHAPI_HD_daynight_final.html – Interactive map showing daytime and nighttime Spatial Heat Adaptation Priority Index (SHAPI) classes.
- SHAPI_interactive_map.ipynb – Python Jupyter Notebook containing the code used to generate the interactive SHAPI visualization.

**To view the interactive SHAPI map** click on the link _https://rhitujab04.github.io/SHAPI_Interactive_Map/_

The map allows users to toggle between day and night layers with reference locations.
